# 🧑‍💼 HR Management System (MVP)

A simple HR Management System built using **HTML, CSS, and JavaScript**.  
This project allows basic employee management using browser local storage.

---

## 🚀 Features

- 🔐 Simple Login System (Admin only)
- 🧑‍💼 Add Employees
- ✏️ Edit Employee Details
- ❌ Delete Employees
- 🔍 Search Employees
- 💾 Data stored in browser (LocalStorage)

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap (for UI styling)
